
Minecraft-Style PHP UI (5 files)

Files:
- index.php         -> Home screen (Minecraft-style)
- play.php          -> Wraps your CRUD app (includes 'crud.php' by default)
- achievements.php  -> Placeholder
- marketplace.php   -> Placeholder
- options.php       -> Placeholder
- partials/header.php & partials/footer.php
- assets/styles.css

How to use:
1) Put all these files on your PHP server (same folder).
2) Rename your existing CRUD entry file to 'crud.php' and place it in the same folder.
   OR access: play.php?file=YOURFILENAME.php
3) Open index.php in your browser.
4) "Quit Game" shows a Goodbye message and attempts to close the tab (may be blocked).

Font:
- Uses Google Font "Press Start 2P" to emulate pixel UI.

Version:
- Bottom left shows 01.00.00.01
